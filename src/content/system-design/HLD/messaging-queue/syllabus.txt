16️⃣ Kafka vs RabbitMQ Decision Framework

You must know when to use what.

Use Case	RabbitMQ	Kafka
Task Queue	✅ Best	❌ Not ideal
Event Streaming	❌ Limited	✅ Best
High Throughput	⚠️ Moderate	✅ Very High
Strict Ordering	Per queue	Per partition
Long-term storage	❌ No	✅ Yes

If you can explain this confidently → Lead level.
