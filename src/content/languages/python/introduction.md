# Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular languages for beginners and professionals alike.

## Why Learn Python?

Python is an excellent choice for both beginners and experienced developers:

- **Easy to Learn**: Simple syntax that reads like English
- **Versatile**: Used in web development, data science, AI, automation, and more
- **Large Community**: Extensive libraries and frameworks
- **High Demand**: One of the most sought-after skills in tech

## Real-World Examples

### Example 1: Hello World

```python
print("Hello, World!")
```

That's it! Python's simplicity makes it easy to get started.

### Example 2: Simple Calculator

```python
# Simple calculator
a = 10
b = 5

print(f"Sum: {a + b}")
print(f"Difference: {a - b}")
print(f"Product: {a * b}")
print(f"Quotient: {a / b}")
```

Notice how Python doesn't require type declarations - it's dynamically typed!

### Example 3: Working with Lists

```python
# Shopping list
shopping_list = ["apples", "bananas", "milk", "bread"]

print("Shopping List:")
for item in shopping_list:
    print(f"- {item}")

# Add an item
shopping_list.append("eggs")
print(f"\nUpdated list: {shopping_list}")
```

## Key Features

- **Interpreted Language**: No compilation needed
- **Dynamic Typing**: Variables don't need type declarations
- **Indentation-based Syntax**: Code blocks are defined by indentation
- **Rich Standard Library**: Many built-in modules for common tasks

## Common Use Cases

1. **Web Development**: Django, Flask frameworks
